📝模块 3：核心知识点精编 ▼

P0GDP的三种核算方法

来源： Slide 3-5

GDP 是指经济社会在一定时期内运用生产要素所生产的全部最终产品（物品和劳务）的市场价值。

通俗理解：一个国家一年内"造出来的所有东西的总价"，只算最终产品，中间产品不计入。

考试形态

题型常见问法
选择下列哪项不计入 GDP？
简答简述 GDP 三种核算方法及其关系
计算给定数据用支出法计算 GDP

易错警示

  • ❌ GDP vs GNP：GDP看"在哪里生产"，GNP看"谁生产的"
  • ❌ 二手交易：二手商品不计入当年GDP（已计入生产年份）
  • ❌ 存货投资：生产但没卖出去的也算在GDP中

P0乘数效应

来源： Slide 12-14

乘数（Multiplier）是指自发总支出的增加所引起的国民收入增加的倍数。

核心公式

\[ k = \frac{\Delta Y}{\Delta A} = \frac{1}{1-MPC} \]
乘数类型公式说明
投资乘数\(k_i = 1/(1-MPC)\)MPC = 边际消费倾向
政府支出乘数\(k_G = 1/(1-MPC)\)与投资乘数相同
税收乘数\(k_T = -MPC/(1-MPC)\)绝对值小于支出乘数
平衡预算乘数\(k_B = 1\)G = T 时成立

易错： \(k_T = k_G - 1\)（税收乘数永远比政府支出乘数小1）

P0IS-LM模型

来源： Slide 18-22

IS曲线：产品市场均衡（投资=储蓄），利率与产出负相关。
LM曲线：货币市场均衡（货币需求=货币供给），利率与产出正相关。
\[ \text{IS: } Y = C(Y-T) + I(r) + G \qquad \text{LM: } \frac{M}{P} = L(Y,r) \]

考试要点：IS-LM交点 = 产品市场+货币市场同时均衡。财政政策移动IS，货币政策移动LM。

GDP vs GNP

维度GDPGNP
定义一国境内生产的最终产品价值一国国民生产的最终产品价值
原则国土原则国民原则
关系\(GNP = GDP + \text{NFP}\)（国外净要素收入）

名义GDP vs 实际GDP

维度名义GDP实际GDP
价格当期价格基期价格
变化原因产量变化 + 价格变化仅产量变化
关系\(名义GDP = 实际GDP \times GDP平减指数\)

选择题

Q1. 下列哪项不计入 GDP？

  • A. 政府购买新办公楼的支出
  • B. 二手车的交易价格
  • C. 企业存货的增加
  • D. 出口到国外的商品

答案：B

解析：二手交易不产生新价值，已计入生产年份的GDP。

简答题

Q2. 简述乘数效应的作用机制，并说明为什么税收乘数绝对值小于支出乘数。

答题要点：

  • 自发支出增加 → 收入增加 → 消费增加 → 收入进一步增加（循环）
  • 税收乘数：增税 → 可支配收入减少 → 消费减少（间接）
  • 支出乘数：政府直接购买 → 直接增加总需求（直接）
  • 所以 \(|k_T| < k_G\)

计算题

Q3. 已知 C = 100 + 0.8Yd, I = 50, G = 200, T = 0.25Y, 求均衡国民收入。

解题思路：

  1. Yd = Y - T = 0.75Y
  2. C = 100 + 0.8(0.75Y) = 100 + 0.6Y
  3. Y = C + I + G = 100 + 0.6Y + 50 + 200
  4. 0.4Y = 350 → Y = 875
