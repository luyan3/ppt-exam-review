## 示例 2：核心知识点精编（节选）

```markdown
### [#1] GDP 的三种核算方法 🔴 P0
**来源：** P3-P5

**📖 定义（原文）**
> GDP（国内生产总值）是指经济社会（一国或一地区）在一定时期内运用生产要素所生产的**全部最终产品（物品和劳务）的市场价值**。

**💡 通俗理解**
一个国家一年内"造出来的所有东西的总价"，且只算最终产品（中间产品不计入，避免重复计算）。

**📌 考试形态**
| 题型 | 常见问法 | 频次 |
|------|---------|------|
| 选择 | 下列哪项不计入 GDP？ | 高频 |
| 简答 | 简述 GDP 三种核算方法及其关系 | 高频 |
| 计算 | 给定数据用支出法计算 GDP | 中频 |

**⚠️ 易错警示**
- ❌ 混淆 GDP vs GNP：GDP 看"在哪里生产"，GNP 看"谁生产的"
- ❌ 误记二手交易：二手商品交易不计入当年 GDP（已计入生产年份）
- ❌ 漏记存货投资：生产出来但没卖出去的也算在 GDP 中（存货投资）

---

### [#4] 乘数效应 🔴 P0
**来源：** P12-P14

**📖 定义（原文）**
> 乘数（Multiplier）是指自发总支出的增加所引起的国民收入增加的倍数。$k = \frac{\Delta Y}{\Delta A}$

**💡 通俗理解**
政府花 1 块钱 → 变成某些人的收入 → 他们再花一部分 → 循环往复 → 最终 GDP 增长可能超过 1 块。

**📐 核心公式**
| 乘数类型 | 公式 | 说明 |
|---------|------|------|
| 投资乘数 | $k_i = \frac{1}{1-MPC}$ | MPC=边际消费倾向 |
| 政府支出乘数 | $k_G = \frac{1}{1-MPC}$ | 与投资乘数相同 |
| 税收乘数 | $k_T = -\frac{MPC}{1-MPC}$ | 绝对值小于支出乘数 |
| 平衡预算乘数 | $k_B = 1$ | 政府支出=税收时，乘数为1 |

**⚠️ 易错警示**
- ❌ 乘数作用双向：减少支出同样会被放大（经济衰退时更明显）
- ❌ 税收乘数为负：增税 → 可支配收入↓ → 消费↓ → Y↓
- ✅ 记忆：$k_T = k_G - 1$（税收乘数永远比政府支出乘数小 1）
```

## 示例 3：一页纸终极总结

```markdown
# 宏观经济学 — 考前速览

## 🔴 必考核心
1. **GDP 核算** — 支出法 Y=C+I+G+NX，收入法、生产法 → P3
2. **乘数效应** — $k=1/(1-MPC)$，双向作用，$k_T=k_G-1$ → P12
3. **IS-LM 均衡** — 产品市场+货币市场同时均衡 → P18-P22
4. **AD 曲线** — 向右下倾斜（财富效应、利率效应、汇率效应）→ P25
5. **菲利普斯曲线** — 短期:通胀↑失业↓；长期:垂直（自然失业率）→ P35

## 🟡 重要概念
- **GDP vs GNP**: 地域 vs 国民 → P3
- **名义 vs 实际 GDP**: 前者用当期价格，后者用基期价格 → P6
- **消费函数**: $C=\alpha+\beta Y$，$\beta=MPC$ → P10
- **流动性偏好**: 交易动机+预防动机+投机动机 → P17
- **通胀类型**: 需求拉上型 vs 成本推动型 → P33

## 📐 关键公式
| 公式 | 说明 | 页 |
|-----|------|----|
| $Y=C+I+G+NX$ | 支出法恒等式 | P4 |
| $k=1/(1-MPC)$ | 乘数 | P12 |
| $IS: Y=C(Y-T)+I(r)+G$ | 产品市场均衡 | P18 |
| $LM: M/P=L(Y,r)$ | 货币市场均衡 | P20 |
| $PC: \pi=\pi^e-\beta(u-u^n)+v$ | 菲利普斯曲线 | P36 |

## ⚠️ 注意混淆点
1. IS 左移 vs LM 右移 — 效果看哪个影响更强
2. 短期AS水平 vs 长期AS垂直 — 价格黏性假设不同
3. 结构性失业 vs 摩擦性失业 — 前者技能不匹配，后者找工作时间
```
